## Dataset Overview

### Dataset Parameters

**Electrical Parameters**
- vdc1
- vdc2
- idc1
- idc2

**Environmental Parameters**
- irr : Irradiance
- pvt : PV Temperature

**Fault Label**
- fault

---

### Fault Label Description

| Label | Fault Type                          |
|------:|-------------------------------------|
| 0     | Normal Operation (No Faults)         |
| 1     | Short Circuit                        |
| 2     | Degradation                          |
| 3     | Open Circuit                         |
| 4     | Shadowing                            |

---

### Notes
- Dataset is stored in MATLAB (.mat) format.
- File size is approximately 27 MB.
- Due to GitHub file size limitations, the dataset is not uploaded.
